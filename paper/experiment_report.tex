\section{Method Overview}

We extend the Deep Emulator (Zheng et al., CVPR 2021) with:
\begin{enumerate}
\item \textbf{Multi-Scale Edge Aggregation (MSEA)}: multi-hop graph message passing
\item \textbf{Physics-informed features}: relative displacement, reference velocity, $\kappa/m$ ratio
\item \textbf{Causal Cone Masking}: learned per-vertex propagation speed gates spatial scale access
\item \textbf{Residual Attention}: 80\% cone-gated + 20\% unmasked bypass for robust information flow
\end{enumerate}

\section{Key Findings}

\begin{enumerate}
\item \textbf{19\% average improvement over baseline at E=250K--8M} across 6 motions, winning on all 13 stiffness values in this range.

\item \textbf{Causal cone contributes 9\% improvement} over the same architecture without cone (ablation).

\item \textbf{Residual attention bypass} (20\% unmasked) improves win rate from 42\% to 56\% by allowing information to flow past the cone when needed.

\item \textbf{Physics-informed features are critical}: relative displacement, velocity, $\kappa/m$ ratio reduced the gap from 11$\times$ to 1.4$\times$ vs baseline.

\item \textbf{Param count is not the advantage}: wide baseline (528K params) performs identically to original (237K).

\item \textbf{Results are reproducible}: std = 0.000024 across 3 seeds (9\% CV).

\item \textbf{Rollout is stable}: 1.1--1.5$\times$ baseline (vs 100$\times$ in early versions).
\end{enumerate}
